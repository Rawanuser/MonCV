Ce que je fais

UI Design

Le UI design est un processus de conception centré sur l'utilisateur qui consiste à concevoir l'aspect et l'expérience utilisateur d'une application ou d'un site web. Il s'agit de créer une interface utilisateur esthétique, conviviale et facile à utiliser, en prenant en compte les besoins et les attentes des utilisateurs finaux.

Développement web

Le développement web (Web Developement) est le processus de création de sites web et d'applications web. Cela implique la conception, le codage, la programmation et la mise en œuvre de fonctionnalités et de contenus pour un site web ou une application.

Design graphique

Le design graphique est le processus de création visuelle utilisant des éléments tels que la typographie, les images, les formes et les couleurs pour communiquer un message ou une idée. Le design graphique est utilisé dans une variété de domaines, tels que la publicité, l'édition, la création de marques et la conception de sites web.
